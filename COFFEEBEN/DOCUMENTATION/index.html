Description

COFFEEBEN is a beautiful easy to use coffee shop and pastry template built on bootstrap 4, Its can be used especially for modern coffee shop.The template is responsive, so you can view it also in the mobile/tablets devices and it looks very nice in thoose devices. COFFEEBEN is retina ready that means it looks great and very sharp in high resolution devices.

The template have :

  1. Responsive layout
  2. Retina Ready
  3. Multiple section to feat every needs
  4. One page navigation
  5. Html 5 & CSS3
  6. Unique design
  7. Block design sections
  8. Modern design
  9. Built on bootstrap 4

Installation

When you download the main download file you finde several folder  :

  •       HTML folder this is the folder that contain all the filse of the template
  •       LICENSES folder this folder contain the licenses of the elements used.
  •       DOCUMENTATION : this folder contain the documentation of the theme please read the documentation carefully for further information

Icons

COFFEEBEN  uses fonttello icons and Flaticon , the template come with a set of icons that you can use as you want, and change the default icons to fit your needs.You can find the list of the icons in the fonts.css in the css folder then you can choose the icon that you want and change it in the html code.

 

Flaticon has been used for demo purposes only and are not included in the main download file, but you can use them without any problems by your own,

Map setup

COFFEEBEN come with a map location section to locate multiple locations in the  google map. To setup your coordinate open script.js file then scroll down until you find the map script setup. You are concerned with lines od code below :
var center = {
        lat: 39.138038,
        lng: -84.339744
    };

    var locations = [
        ['Newyork<br>\ 63 W 38th St, New York NY 10018<br>\
   <a href="#">Get Directions</a>', 39.137283, -84.317716],

        ['Brooklyne<br>\ 1080 Broadway, Brooklyn, NY 11221<br>\
   <a href="#">Get Directions</a>', 39.145362, -84.374428],

        ['Chicago<br>\ 225 N Wabash Ave, Chicago, IL 60601<br>\
   <a href="#">Get Directions</a>', 39.138038, -84.339744]

    ];

To setup you locations in the map jus replace the current coordinates with yours.

The center variables setup the center postion of the map you can replace this variable as well with your initial position coordinates.

Key map setup

The map setup also require a specific key your own key that you put along when you calling the google map api as below :

 

<script src="https://maps.googleapis.com/maps/api/js?key=PUTYOURMAPKEYHERE&callback=initializeMap"></script>

 

You can find the line above in the index.html file at the bottom.

 

You can get the key here : https://developers.google.com/maps/documentation/javascript/get-api-key
